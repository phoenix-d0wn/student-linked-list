## What I Learned ✏️
#### 1. (Singly, Non-Circular) Linked Lists using Pointers in C++
  - How pointers work, generally
  - How to insert and delete nodes in a linked list
  - How to traverse a linked list
#### 2. File Handling in C++
  - How to read from and write to external files
#### 3. Basic Use of Makefile
  - How to use a Makefile to compile the program more easily
#### 4. Coding Workflow
  - Testing robustness and functionality before moving on to the next step

## What Could Be Improved 🧠
#### 1. Student Matric No. & IC No. Validation
  - No two or more students should share the same Matric # or IC #
  - Matric # and IC # should not exceed a certain number of characters and must follow a strict format
#### 2. Prompt User to Enter Passport # (if Applicable)
  - For international students
#### 2. Sort List
  - Sort list by Matric #/IC #/Name
#### 3. Student Name, Program, & CGPA Validation
  - Student name should not exceed a certain number of characters
  - Program for student should be from a select few names, e.g. BAXI and BAXZ only
  - Ask User to retype CGPA value if CGPA < 0.00 or CGPA > 4.00
