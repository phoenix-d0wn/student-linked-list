# student-linked-list
A repository for BAXU1143 Data Structure &amp; Algorithm, Lab Assessment 2.

---
## How to Install & Run ⚙️
#### 1. Open Terminal in Project Directory
#### 2. Run 'make' in the Terminal
#### 3. Run './app' in the Terminal
--
## Features ⭐

#### 1. Add New Student Record
#### 2. Delete Student Record
#### 3. Search Student Record using Matric # (Student ID)
#### 4. Search Student Record using IC #
#### 5. Search Student Record(s) using Program
#### 6. Display All Student Records
#### 7. Calculate Average Student CGPA

--
## What I Learned ✏️
#### 1. (Singly, Non-Circular) Linked Lists using Pointers in C++
  - How pointers work, generally
  - How to insert and delete nodes in a linked list
  - How to traverse a linked list
#### 2. File Handling in C++
  - How to read from and write to external files
#### 3. Coding Workflow
  - Testing robustness and functionality before moving on to the next step
